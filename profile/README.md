## 加入我们
我们需要翻译、时轴、压制等人员共同努力，制作更多、更好的作品

欢迎你的加入

### 入前须知
* 决定加入前请先确定本组的制作理念，避免与自己的理念不合
* 请先确定自己有一定时间分配给业余爱好，拒绝三分钟热度，做字幕本身枯燥乏味，请确定能在自己的职位上制作至少半季番
* 本组提供除翻译外的其他技术教学，但仅接受有一定自学能力的爱好者
